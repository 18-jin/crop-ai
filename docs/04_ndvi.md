# 04 | NDVI、Togo、以及你问的三个核心问题

## 1) 这张图“抽的是哪张图片/哪个位置”？

你画出来的图 **不是从某一张卫星图片裁出来的**。

它是：
- 从 **Togo benchmark 数据集**里拿到的一批样本（点/小区域）
- 对每个样本，已经有 **12 个时间步 × 18 个 band** 的特征
- 你选其中的 **NDVI band**（18 个里最后一个）
- 分别对 `y=1`（Crop）与 `y=0`（Non-crop）做平均
- 得到两条“季节曲线”

所以：这张图表示的是 **“一堆样本的平均时序特征”**，不是一个地块的影像截图。

## 2) 为什么 NDVI 会出现负数？（不是“负概率”）

NDVI 不是概率，它是一个指数：

> NDVI = (NIR - Red) / (NIR + Red)

它的取值范围是 **[-1, 1]**。

一般经验：
- NDVI < 0：多为水体、云、阴影、雪、建筑等（红光比近红外更强/或近红外很弱）
- 0 ~ 0.2：裸地/稀疏植被
- 0.2 ~ 0.5：一般植被
- 0.5+：茂密植被（作物旺盛期可能更高）

你现在画的曲线有负数，常见原因：
- 数据做了归一化/标准化（mean/std）后，数值不再是原始 NDVI
- 或者该数据版本里 NDVI 的工程化定义/裁剪方式导致范围变化
- 或样本含有很多“非植被像元”（道路、裸土、水体）

> 关键：**负数不是错，也不是概率。** 是指数数值，代表“近红外相对红光没有那么强”。

## 3) Togo 是什么？NDVI 是什么？

- **Togo（多哥）**：西非国家名。CropHarvest benchmark 里有一个“多哥任务”。
- **NDVI**：归一化植被指数，是遥感里最常用的植被活力指标之一。

## 4) 你对“近红外/红光/绿色植被”的理解：哪些对、哪些需要纠正

你说：
> 近红外就是能产生更高温度，传播更远，波长更大的红光吧

需要纠正：
- 近红外（NIR）是 **电磁波波段**，波长比可见光红光更长，但它不是“更热/更远”这么理解。
- “温度”更多对应 **热红外（TIR）** 的辐射，不是近红外。

你说：
> 绿色植被增多导致红光反射率变低

这句直觉是对的（更准确说）：
- 绿叶的叶绿素吸收可见光（尤其红光）用于光合作用 → 红光反射低
- 叶片内部结构对近红外反射强 → NIR 反射高
→ 所以 NDVI 上升。

你说：
> 如果纬度增高，成熟期 NDVI 掉的会比这个更多

这是“可能成立”的直觉：
- 高纬度季节性更强、冬季更长，植被周期更剧烈
- 但具体还受作物类型、灌溉、种植制度、云量影响
